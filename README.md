# NewsApp

A React-based news application that displays top headlines from various categories using the NewsAPI.

## Features

- Browse news by categories: General, Business, Entertainment, Health, Science, Sports, Technology
- Infinite scroll for seamless news browsing
- Top loading bar for better UX
- Responsive design with Bootstrap

## Technologies Used

- React 19
- React Router v7
- NewsAPI
- React Infinite Scroll Component
- React Top Loading Bar
- Bootstrap

## Setup

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser
